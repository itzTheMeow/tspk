# Tspk

_(pronounced T-Speak)_

A simple Text-To-Speech/Soundboard in your terminal.

## Getting Started

Install dependencies using pnpm.

```bash
pnpm install
```

## Running

To run the app run the start script.

```bash
pnpm start
```

## Development

All config keys should be lowercase.
